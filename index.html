Embedded multicore image decoder

On an ALtera FPGA board, this system decodes compressed 640x480 mpeg video at 24fps. An IDCT hardware accelerator, written in VHDL, handles the most computationally intensive function. Developed at the University of Waterloo for ECE423: Embedded Computer Systems.

Real-time OS

A single-threaded multiprogramming operating system that runs on an ARM microprocessor. Performs memory management, process dispatching and switching, interprocess communication, UART and timer interrupts and runs user processes using a custom command set. Developed at the University of Waterloo.
