Projects

Embedded multicore image decoder

On an ALtera FPGA board, this system decodes compressed 640x480 mpeg video at 24fps. An IDCT hardware accelerator, written in VHDL, handles the most computationally intensive function.

Superpixel algorithm implementation

Superpixels are used as a precursor step to image segmentation. This project implements a state-of-the-art superpixel algorithm based on current image science literature.

Real-time OS

A single-threaded multiprogramming operating system that runs on an ARM microprocessor. Performs memory management, process dispatching and switching, interprocess communication, UART and timer interrupts and runs user processes using a custom command set.

Run Tracker (currently in alpha)

Run Tracker saves your paths, generates useful statistics, and runs on all Apple and Android smartphones. Using Google Maps APIs, this is a useful and powerful mobile web application.
